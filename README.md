## Greetings

- Formal expressions
  - Good morning
  - Good afternoon
  - Good evening
  - Good night

- Informal expressions
  - Hi
  - Hello
  - How are you?

  - And you?

  - How are you doing?
  - How is it going?
  - Hi, how is it doing?
  - Hey, what's up
